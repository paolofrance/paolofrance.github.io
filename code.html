to download useful basic code, please follow this installation
follow the manual installation procedure until "workspace 2: control repository"
at this point, please do the following to install the packages to use the UR5 and the game-theoretical controllers 
$ mkdir -p ~/gt_ws/src
$ cd ~/gt_ws
$ catkin init
$ wstool init src
$ mkdir rosinstall
$ wget https://raw.githubusercontent.com/paolofrance/paolofrance.github.io/master/rosinstall/gt_ws.rosinstall  -P ./rosinstall 
$ wstool merge -t src ./rosinstall/gt_ws.rosinstall 
$ wstool update -t src
$ rosdep install --from-paths src --ignore-src -r -y
$ catkin config -j 2
$ catkin build -cs --mem-limit 30%
